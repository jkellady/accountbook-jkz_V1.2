How to generate icons

  1. Right-click each canvas and choose "Save image as...", OR click the download buttons.
  2. Save all 4 PNG files to public/icons/.
  3. The manifest at app/manifest.ts references these files.
  4. These icons are used by iOS, Android, and desktop browsers for the install prompt and home screen.
